# Privacy Policy

YouTube Auto Transcript TXT does not collect, store, sell, or transmit user data to the developer or to any third-party service.

## Data Access

The extension can access the active YouTube tab only after the user clicks the extension icon. It reads YouTube page data and caption metadata needed to create a local transcript file.

## Data Transfer

Transcript data is fetched from YouTube using the browser session for the current page. The extension does not send transcript text, page URLs, browsing history, account data, or identifiers to any non-YouTube service.

## Storage

The extension does not use Chrome storage and does not persist transcript content.

## Downloads

The extension creates a local `.txt` file by generating a Blob URL in the current page. It does not use the Chrome `downloads` permission.
